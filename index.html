Regras

  • 1. Início: O Mestre gera os IDs e distribui para os convidados. Cada ID gera uma cartela única e fixa.
  • 2. Marcação: Toque no número para colocar um grão de milho. Para remover, toque novamente e confirme.
  • 3. Linha: Se completar uma linha horizontal, vertical ou diagonal (5 números), você bateu LINHA. Grite para ganhar o prêmio de consolação!
  • 4. Bingo: Se completar a CARTELA TODA (24 números + livre), grite BINGO e mostre o celular piscando para o Mestre.
  • 5. Fraude: O Mestre tem um radar de todas as cartelas. Se gritar falso, a punição é um shot de bebida barata.
